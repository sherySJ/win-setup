SFC (System File Checker) Process
---------------------------------
1. Step 1: Run SFC /Scannow
   sfc /scannow

2. Step 2: If Errors Found, Run DISM
   DISM /Online /Cleanup-Image /RestoreHealth

3. Step 3: Run SFC /Scannow Again
   sfc /scannow
